# Hot-Keywords-Extractor-
A simple web application that extracts text from any webpage URL and analyzes it to find the most frequent and hot keywords using NLP techniques.
##  Features
- Fetches and scrapes webpage content using `Requests` and `BeautifulSoup`.
- Cleans extracted text (HTML tags, punctuation, stopwords).
- Applies NLP processing (Tokenization, Lemmatization).
- Extracts the most frequent words (Hot Keywords).
- Interactive frontend built with **HTML, CSS, JavaScript**.
- Backend API built using **Flask**.

## Technologies Used
- **Python**
- **Flask**
- **BeautifulSoup**
- **Requests**
- **NLTK / spaCy**
- **HTML, CSS, JavaScript**
